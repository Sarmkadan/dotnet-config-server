# Security Policy

We take the security of this project seriously. If you discover any security-related issues, please follow this responsible disclosure policy.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities. Instead, you can report them securely by using one of the following methods:

1. Use GitHub's built-in **private advisory feature** in this repository.
2. Contact us directly via email at **rutova2@gmail.com**.

## Response Timeline

We will provide an **acknowledgment within 48 hours** of receiving your report.

## Supported Versions

Currently, the following versions are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |
